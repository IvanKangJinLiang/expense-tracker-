# Expense Tracker (Laravel + Vue 3)

A full-stack Expense Tracker application built with **Laravel 10/11** (Backend) and **Vue 3** (Frontend). It features full CRUD functionality, real-time validation, and currency formatting.

## Prerequisites

Before you begin, ensure you have the following installed on your machine:
* **PHP** (8.1 or higher)
* **Composer** (PHP Dependency Manager)
* **Node.js & NPM** (JavaScript Runtime)
* **MySQL** 

## Installation Guide

Follow these steps to set up the project locally.

### 1. Clone the Repository

```bash
git clone <YOUR_REPO_URL_HERE>
cd expense-tracker
```

### 2. Install Backend Dependencies (PHP)
```bash
composer install
```

### 3. Install Frontend Dependencies (Vue)
```bash
npm install
```
